# Collection Validation Attributes

This folder contains attributes for validating collections in the EasyValidate library.

## Attributes

### LengthAttribute
Ensures that a collection has a length within a specified range.

### MaxLengthAttribute
Ensures that a collection does not exceed a specified maximum length.

### MinLengthAttribute
Ensures that a collection has at least a specified minimum length.

### ContainsAttribute
Ensures that a collection contains a specific value.

### DoesNotContainAttribute
Ensures that a collection does not contain a specific value.

### AllMatchAttribute
Ensures that all items in a collection match a specified condition.

### DistinctByAttribute
Ensures that a collection has unique items based on a specified property or key.
