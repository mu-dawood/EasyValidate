# Installation

Install EasyValidate from NuGet:

```shell
Install-Package EasyValidate
```

Or with .NET CLI:

```shell
dotnet add package EasyValidate
```

Add the analyzer package for compile-time validation:

```shell
Install-Package EasyValidate.Analyzers
```
