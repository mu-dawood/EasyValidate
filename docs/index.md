# EasyValidate

Welcome to the EasyValidate documentation!

## Overview
EasyValidate is a .NET validation library and Roslyn analyzer package for attribute-based validation. It helps you write clean, maintainable, and robust validation logic for your models and DTOs.

## Repository
[GitHub: mu-dawood/EasyValidate](https://github.com/mu-dawood/EasyValidate)

## Getting Started
- Install the NuGet packages:
  - `EasyValidate`
  - `EasyValidate.Analyzers`
- Add validation attributes to your models.
- Use the generated `Validate()` methods or validation results in your code.
