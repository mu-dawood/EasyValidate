## 🚀 Features
- Modern attribute-based validation for .NET
- Source generator for fast, compile-time validation
- Roslyn analyzers for best practices and error prevention
- Easy integration with your models and DTOs

## 📦 Installation
Install via NuGet:
```shell
# Library
Install-Package EasyValidate
# Analyzers (optional, but recommended)
Install-Package EasyValidate.Analyzers
```
